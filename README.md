# Sales Performance Analysis & Forecasting

## Description
A Python-based analytics project that analyzes historical sales data, generates business insights, and forecasts future sales trends. This project demonstrates analytical thinking, basic statistical modeling, documentation, and testing aligned with analyst roles.

## Features
- Load and process CSV sales data
- Calculate total and average sales
- Generate monthly sales insights
- Forecast future sales using trend-based logic
- Validate logic using test cases

## Technologies Used
- Python
- CSV File Handling
- Basic Statistics

## How to Run
Run the program using:
python sales_analysis.py
